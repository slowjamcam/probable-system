PASSWORD MANAGER

A password manager that allows you to save your own password. A password manager that allows you to save your own password and also creates a "Very Strong" password for any username, email, or account.
Also is saved locally on your PC so no worries about hackers getting to your passwords!
The app allows the user to create a password for any length necessary!

This project was made over the span of about a couple of months, and time spent coding was about a week. There was a lot of research that went into this project. It is one of my most used projects, but there are more features that I want to implement!
I am progressively researching into bettering the security of the project and the ease of use on the client's side.
